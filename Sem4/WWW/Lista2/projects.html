Smocze Skarby (C++) pdf

Moim zadaniem było napisanie programu na konkursie CMI, który znajdzie współrzędne jamy smoka (opis w pdf) na mapie, wykonując N kroków zgodnie z opisem w manuskrypcie.

Opis wejścia: W pierwszym wierszu znajduje się liczba T (1 ≤ T ≤ 100 000) oznaczająca liczbę przypadków testowych. Każdy z kolejnych T wierszy zawiera liczbę całkowitą N (0 ≤ N ≤ 10^18) – liczbę kroków do wykonania.

Opis wyjścia: Na wyjściu należy wypisać T wierszy, każdy zawierający współrzędne (x, y) smoczej jamy dla danego przypadku testowego.

Przykład:

4
            3
            0
            30
            8

Odpowiedź:

1 0
            0 0
            -5 -3
            2 -2

Spektakularna część kodu

void zwracaniedwojki(long long N, int xnew, int ynew) {
                    if (N == 3) {

                            dwojki.push_back(make_pair(xnew, ynew));
                        }
                        long long x, y;
                        x = 0 * (0 - xnew) - 1 * (0 - ynew) + xnew;
                        y = 1 * (0 - xnew) + 0 * (0 - ynew) + ynew;
                        dwojki.push_back(make_pair(x, y));
                        licznik3 *= 2;
                        if (licznik3 < N) {
